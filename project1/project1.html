SEO Best Prectices

How Search Engines Work

Search engines work by crawling, indexing, and ranking web content. Crawlers, sometimes called spiders, follow links from page to page to discover documents on the web. After a page is found, selected information is stored in an index. When a user searches for information, the search engine returns results based on relevance, usefulness, popularity, and other ranking factors.

Importance of Keywords and Description Meta Tags

Keywords and description meta tags help explain what a page is about. Keywords should match the words people are likely to use when searching, while the description meta tag provides a short summary of the page. These tags should be written for humans first because clear, accurate descriptions can help users decide whether the page meets their needs.

Four Elements of Relevant Content Links

Relevant content links should include descriptive link text, a useful destination, a clear relationship to the surrounding content, and accessibility for users and search engines. Strong links should avoid vague phrases such as “click here.” Instead, the linked words should explain what the user will find, such as a project page, gallery, resource, or related topic.

Good Heading Hierarchy

Good heading hierarchy means organizing headings in a logical order. A page should have one h1 heading that identifies the main topic. Major page sections should use h2 headings, and smaller subsections should use h3 or lower headings. For example, “CMST 386 - Project 1” is the h1, “SEO Best Practices” is an h2, and “How Search Engines Work” is an h3.

Heading Example

A clear structure helps visitors scan the page and helps assistive technology communicate the page organization. It also supports SEO because search engines use headings as clues about the page content.

User Engagement and Accessibility

A well-designed website should be useful, readable, and easy to navigate. This includes using semantic HTML, descriptive page titles, meaningful link text, readable color contrast, and accurate alternative text for images. These choices help people understand the page while also giving search engines better information about the content.
